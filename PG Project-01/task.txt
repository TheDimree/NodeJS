REST API - JSON
Data is used from: https://www.mockaroo.com/


GET /users - HTML document renders.
GET /api/users - List all users JSON - Done

GET /api/users/1 - Get the user with id = 1
GET /api/users/2 - Get the user with id = 2

POST /api/users - Create new user

PATCH /api/users/id - Edit the user with given id

Delete /api/users/id - Delete the user with given id

Dynamic Path Parameters
GET /api/users/:id = colon means this is a variable.
